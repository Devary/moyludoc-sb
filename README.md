# moyludoc-sb

Separate Spring Boot migration target for Moyludoc.

## Stack
- Spring Boot
- Thymeleaf (Qute replacement)
- Same document parsing/preview services migrated from Quarkus

## Run
```bash
./mvnw spring-boot:run
# or
mvn spring-boot:run
```

Open: <http://localhost:8081/api/docx/library/browser>
